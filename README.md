# React + PHP アプリケーション

このリポジトリは、React をフロントエンドに、PHP をバックエンドとして構成された Web アプリケーションです。UIフレームワークとして Bootstrap を利用しています。

## 📦 技術スタック

- **フロントエンド**: React
- **UIフレームワーク**: Bootstrap
- **一部 TypeScript にて記述**
- **バックエンド**: PHP  
  - ※ Laravel も検討しましたが、レスポンスの軽量さを優先し、素の PHP にて構成しています  
  - 将来的には Express（Node.js）への移行を想定

## 🛠️ 開発の背景と方針

- 現時点では JavaScript 中心で構成されていますが、一部 TypeScript を導入済み
- 今後のリファクタリングにおいて、TypeScript への全面移行を進め、型安全性と可読性の向上を目指します
- サーバーサイドも将来的に Node.js（Express）への置き換えを視野に入れています
